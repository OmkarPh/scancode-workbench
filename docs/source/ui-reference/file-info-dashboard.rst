============================
:index:`File Info Dashboard`
============================

Scan must have ``--info`` option for File Info Dashboard

Summary of files:

- Count of
    - Files
    - Directories.

- Pie charts
    - Programming languages
    - File Types
    - Mime Types

Tip - Slices of pie-chart can be hidden/shown by clicking on the corresponding legend below charts
