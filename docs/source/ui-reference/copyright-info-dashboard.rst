===================================
:index:`Copyright Info Dashboard`
===================================

Scan must have ``--license`` option for License Info Dashboard

Summary of Licenses:

- Count of
    - Unique holders
    - Unique notices
    - Unique authors

- Pie charts
    - Copyright Holders
    - Copyright Notices
    - Copyright Authors

Note - Legends are not shown for copyright charts due to large length of copyright statements
