Underlying Technology
=====================

-  ScanCode Workbench is based on `Electron <https://electron.atom.io/>`__ and works on
   Windows, OS X and Linux operating systems.

Platform Support
================

* Linux - ``x64``
* Windows 10/11 - ``x64``
* MacOS - ``x64``, ``arm64``
